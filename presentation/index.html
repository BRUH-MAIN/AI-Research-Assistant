Database Schema & Table Structures

Core User Management

users

  • user_id INTEGER PRIMARY KEY
  • auth_user_id UUID UNIQUE
  • email TEXT NOT NULL UNIQUE
  • first_name, last_name TEXT
  • availability VARCHAR(50)
5 indexes Guest user (ID: 0) AI user (ID: 1)

Group Management

groups

  • group_id SERIAL PRIMARY KEY
  • name TEXT NOT NULL
  • invite_code VARCHAR(12) UNIQUE
  • is_public BOOLEAN
  • created_by INT REFERENCES users

group_participants

  • group_id INT REFERENCES groups
  • user_id INT REFERENCES users
  • role VARCHAR(50) CHECK

Session & Messaging

sessions

  • session_id SERIAL PRIMARY KEY
  • group_id INT REFERENCES groups
  • topic TEXT
  • status VARCHAR(50) CHECK

messages

  • message_id SERIAL PRIMARY KEY
  • session_id INT REFERENCES sessions
  • content TEXT NOT NULL
  • sent_at TIMESTAMP DEFAULT NOW()

Research Papers

papers

  • paper_id SERIAL PRIMARY KEY
  • title TEXT NOT NULL
  • abstract TEXT
  • authors TEXT
  • doi TEXT UNIQUE
  • source_url TEXT

paper_tags

  • paper_id INT REFERENCES papers
  • tag VARCHAR(100)
  • PRIMARY KEY (paper_id, tag)

AI Integration

ai_metadata

  • id SERIAL PRIMARY KEY
  • page_no INTEGER
  • message_id INT REFERENCES messages
  • paper_id INT REFERENCES papers

feedback

  • feedback_id SERIAL PRIMARY KEY
  • session_id INT REFERENCES sessions
  • rating INTEGER CHECK (1-5)
  • content TEXT

Schema Features

  • Normalized Design: 3NF compliance for data integrity
  • Optimized Indexing: Strategic indexes on frequently queried columns
  • Referential Integrity: Foreign key constraints ensure data consistency
  • Check Constraints: Business rule enforcement at database level
  • Default Values: Automatic timestamp and status management

Database Operation Examples

Create Operations

POST /api/groups - Create New Group 
// Express.js Route Handler
router.post('/', async (req, res) => {
  const { name, description, isPublic } = req.body;
  const inviteCode = generateInviteCode();
  
  const { data, error } = await supabase
    .from('groups')
    .insert({
      name,
      description,
      invite_code: inviteCode,
      is_public: isPublic,
      created_by: req.user.user_id
    })
    .select('*')
    .single();
    
  if (error) throw error;
  res.json(data);
});

Read Operations

GET /api/sessions/:groupId - Fetch Group Sessions 
// Complex JOIN query with filtering
const { data, error } = await supabase
  .from('sessions')
  .select(`
    *,
    groups!inner(name),
    session_participants(
      user_id,
      users(first_name, last_name)
    ),
    messages(count)
  `)
  .eq('group_id', groupId)
  .eq('groups.is_public', true)
  .order('started_at', { ascending: false });

PUT /api/users/:id - Update User Profile

Implementation Examples

Live Message Subscription
// Frontend: Subscribe to new messages
const supabase = createClient(url, key);

useEffect(() => {
  const channel = supabase
    .channel(`session-${sessionId}`)
    .on('postgres_changes', {
      event: 'INSERT',
      schema: 'public',
      table: 'messages',
      filter: `session_id=eq.${sessionId}`
    }, (payload) => {
      setMessages(prev => [...prev, payload.new]);
    })
    .subscribe();

  return () => supabase.removeChannel(channel);
}, [sessionId]);
User Presence Tracking
// Track online users in a group
const channel = supabase.channel(`group-${groupId}`, {
  config: { presence: { key: userId } }
});

channel
  .on('presence', { event: 'sync' }, () => {
    const users = channel.presenceState();
    setOnlineUsers(Object.keys(users));
  })
  .on('presence', { event: 'join' }, ({ newPresences }) => {
    // Handle user joining
  })
  .subscribe(async (status) => {
    if (status === 'SUBSCRIBED') {
      await channel.track({ user_id: userId, online_at: new Date() });
    }
  });
Paper Sharing Updates
// Listen for papers added to session
const channel = supabase
  .channel(`session-papers-${sessionId}`)
  .on('postgres_changes', {
    event: '*',
    schema: 'public',
    table: 'session_papers',
    filter: `session_id=eq.${sessionId}`
  }, (payload) => {
    if (payload.eventType === 'INSERT') {
      // Fetch paper details and update UI
      fetchPaperDetails(payload.new.paper_id);
    }
  })
  .subscribe();
